# Oven Spring Bakery - Product Requirements Document

## Project Overview
Mobile-focused, SEO-optimized website for Oven Spring Bakery located in Onehunga, Auckland, New Zealand.

## Key Features Implemented

### 1. Custom Branding (Matching Shop Logo)
- **Logo**: Custom SVG logo matching shop's branding - flour sack with wheat stalk
- **Favicon**: SVG favicon for browser tab matching the sack logo
- **Color Scheme**: Warm cream/beige backgrounds with amber accents and charcoal text

### 2. SEO Optimization (Top Search Results)
- **Meta Tags**: Title, description, keywords for Auckland bakery searches
- **Open Graph**: Facebook/social media preview cards
- **Twitter Cards**: Social sharing optimization
- **Geo Tags**: Location coordinates for local search
- **Schema.org Structured Data**: Full Bakery schema with:
  - Business info (name, address, phone)
  - Opening hours
  - Menu items with prices
  - Aggregate rating
  - Geographic coordinates
- **Semantic HTML**: Proper heading hierarchy (h1-h4)
- **ARIA labels**: Accessibility for screen readers
- **Canonical URL**: Prevents duplicate content issues

### 3. Mobile-First Design
- Responsive layout optimized for mobile devices
- Sticky bottom CTA bar with "Call to Order" and "Directions"
- Touch-friendly navigation and buttons

### 4. Business Information
- **Address**: 164 Trafalgar Street, Onehunga, Auckland 1061
- **Phone**: 022 599 8377 (click-to-call)
- **Instagram**: @oven.spring_ with direct link
- **Rating**: 4.9 stars (110 reviews)
- **Services**: Takeaway & Delivery

### 5. Real Menu Items (from Instagram research)
**Artisan Sourdough Breads**: Country Sourdough, Seeded, Wholewheat, Olive, Ciabatta, Baguette
**Croissants & Viennoiserie**: Butter, Almond, Chocolate, Pain au Chocolat, Ham & Cheese
**Sweet Pastries**: Cinnamon Roll, Morning Bun, Danish, Kouign-Amann, Brioche, Brownie
**Savoury Bakes**: Spinach & Feta Danish, Tomato Tart, Focaccia Sandwich, Sausage Roll, Beef Pie

## Technical Stack
- React (Frontend)
- Tailwind CSS (Styling)
- Lucide React (Icons)
- Custom fonts: Playfair Display (serif), Inter (sans-serif)
